#  Note-App

A simple, modern, and efficient note-taking Android application that allows users to create, edit, and delete notes quickly and easily.

Developed using **Android Studio** and **Java**.

---

##  Features

-  Create new notes  
-  Edit existing notes  
-  Delete notes  
-  Clean and minimal user interface  
-  Fast and smooth performance  

---

##  Technologies Used

- **Android Studio**
- **Java**
- **RecyclerView**
- **Material Design Components**
- **ViewBinding**
